Understanding the display property

The display property does two things. And the two things are: 1) It determine if the box it is applied to acts as inline or block. and 2) The display property also determines how an element's children should behave.

Flexbox and Grid

Flexbox and Grid are the two main layout mechanisms that create layout rules for multiple elements

Flexbox

Flexbox is a layout mechanism for one-dimensional layouts. Layout across a single axis, either horizontally or vertically. By default, flexbox will align the element's children next to each other, in the inline direction, and stretch them in the block direction, so they're all the same height. Flexbox also converts the child elements to be flex items, which means you can write rules on how they behave inside a flex container. The flex property is a shorthand for "flex-grow", "flex-shrink" and "flex-basis".

Grid

Grid is similar in a lot of ways to flexbox, but it is designed to control multi-axis layouts instead of single-axis layouts (vertical or horizontal space). One useful grid unit is the fr unit—which is a fraction of remaining space—you can build traditional 12 column grids, with a gap between each item, with 3 CSS properties. ex: .my-element { display: grid; grid-template-columns: repeat(12, 1fr); gap: 1rem; }

Flow layout

With inline-block you can cause surrounding elements not to respect block margin and padding on an inline element ex: p span { display: inline-block; }

Floats

With floats, if you have an image that sits within a paragraph of text, you can make the text to wrap around the image. ex: img { float: left; margin-right: 1em; }; In this example, the image is instructed to float left, which then allows sibling elements to "wrap" around it.

Positioning

The position property changes how an element behaves in the normal flow of the document, and how it relates to other elements. The available options are "relative", "absolute", "fixed" and "sticky" with the default value being static.ex: .my-element { position: relative; top: 10px; }

When you set position to absolute, it breaks the element out of the current document flow. This means two things:

  • You can position this element wherever you like, using top, right, bottom and left in its nearest relative parent.
  • All of the content surrounding an absolute element reflows to fill the remaining space left by that element.
